# SyncShare - P2P File Transferer

App To directly share file using app

## Installtion Commands

Clone Repository
> git remote add origin git@github.com:sohelshaikh01/React_File_Sharer.git

Navigate to App
> Cd App Name
> Cd backend
> npm install

Install all Dependancies
> npm install

Run Project
> npm run both
